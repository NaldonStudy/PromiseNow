# PromiseNow 시연 시나리오

## 시연 개요
- **서비스명**: PromiseNow (약속 관리 및 실시간 위치 공유 서비스)
- **시연 시간**: 약 15-20분
- **시연 환경**: 웹 브라우저 (Chrome 권장)
- **접속 URL**: https://promisenow.store

## 시연 준비사항
1. **카카오 계정**: 소셜 로그인용
2. **카메라/마이크**: 화상회의 기능 테스트용
3. **위치 권한**: 실시간 위치 공유 기능용
4. **다중 브라우저/탭**: 여러 사용자 시뮬레이션용

---

## 1단계: 서비스 소개 및 로그인 (2-3분)

### 1.1 랜딩 페이지 소개
- "PromiseNow는 약속부터 만남까지 한 화면에서 함께하는 즐거움을 제공하는 서비스입니다."
- "카카오 로그인으로 간편하게 시작할 수 있습니다."

**실행**: 
- 브라우저에서 `https://promisenow.store` 접속
- 랜딩 페이지 UI 요소들 소개
  - 브랜드 로고 및 슬로건
  - 카카오 로그인 버튼
  - PWA 설치 버튼

### 1.2 카카오 소셜 로그인

**실행**:
1. "카카오로 시작하기" 버튼 클릭
2. 카카오 로그인 페이지로 리다이렉트
3. 카카오 계정 정보 입력 및 로그인
4. 권한 동의 (닉네임, 프로필 사진, 이메일)
5. PromiseNow 홈페이지로 자동 리다이렉트

---

## 2단계: 홈페이지 및 방 생성 (3-4분)

### 2.1 홈페이지 소개

**실행**:
1. 사용자 프로필 정보 확인
2. 참여 중인 방 목록 확인 (테스트 데이터)
3. "새로운 방 만들기" 버튼 클릭

### 2.2 방 생성 프로세스

**실행**:
1. 방 제목 입력: "프로젝트 시연 회의"
2. 시작 날짜 선택: 오늘 날짜
3. 종료 날짜 선택: 3일 후
4. "방 만들기" 버튼 클릭
5. 생성된 방으로 자동 이동

---

## 3단계: 방 참여 및 일정 조율 (5-6분)

### 3.1 방 입장 및 참여자 관리

**실행**:
1. 방 제목 및 참여자 수 확인
2. 초대 코드 복사 버튼 클릭
3. 참여자 목록 확인 (본인만 표시)
4. "일정 조율하기" 탭 클릭

### 3.2 일정 조율 기능

**실행**:
1. 캘린더에서 날짜 선택 (오늘 날짜)
2. 시간대별 가능 여부 체크박스 클릭
   - 오전 9시-12시: 가능
   - 오후 2시-6시: 가능
   - 나머지 시간: 불가능
3. "저장" 버튼 클릭
4. 다른 날짜도 선택하여 일정 입력
5. 전체 일정 저장 완료

### 3.3 일정 공유 및 확인

**실행**:
1. 일정 입력 완료 후 결과 확인
2. 겹치는 시간대 하이라이트 표시
3. "일정 추천" 기능으로 최적 시간 제안
4. 확정된 일정 시간 확인

---

## 4단계: 실시간 채팅 기능 (3-4분)

### 4.1 채팅방 입장

**실행**:
1. "채팅" 탭 클릭
2. 채팅 인터페이스 확인
3. 참여자 목록 확인

### 4.2 실시간 메시지 전송

**실행**:
1. 메시지 입력창에 "안녕하세요! 시연 준비되셨나요?" 입력
2. Enter 키 또는 전송 버튼 클릭
3. 메시지가 실시간으로 표시되는 것 확인
4. 추가 메시지 전송: "네, 준비 완료입니다!"
5. 메시지 전송 시간 표시 확인

### 4.3 다중 사용자 시뮬레이션

**실행**:
1. 새 브라우저 창에서 같은 방 URL 접속
2. 카카오 로그인으로 다른 계정으로 로그인
3. 같은 방에 참여
4. 두 사용자 간 실시간 채팅 확인
5. 참여자 목록에 두 사용자 모두 표시되는 것 확인

---

## 5단계: 실시간 위치 공유 (3-4분)

### 5.1 지도 기능 소개

**실행**:
1. "지도" 탭 클릭
2. 지도 인터페이스 확인
3. 현재 위치 권한 요청 확인

### 5.2 위치 공유 설정

**실행**:
1. 위치 권한 허용 버튼 클릭
2. 현재 위치가 지도에 표시되는 것 확인
3. 위치 마커 클릭하여 상세 정보 확인

### 5.3 다중 사용자 위치 확인

**실행**:
1. 다른 브라우저에서도 위치 권한 허용
2. 두 사용자의 위치가 지도에 표시되는 것 확인
3. 위치 마커 간 거리 계산 기능 확인
4. 위치 업데이트 실시간 반영 확인

---

## 6단계: 화상회의 기능 (4-5분)

### 6.1 화상회의 입장

**실행**:
1. "화상회의" 탭 클릭
2. 카메라/마이크 권한 요청 확인
3. 권한 허용 후 비디오 화면 확인

### 6.2 미디어 설정

**실행**:
1. 카메라 ON/OFF 버튼 클릭
2. 마이크 ON/OFF 버튼 클릭
3. 스피커 볼륨 조절
4. 비디오 품질 설정 확인

### 6.3 다중 참여자 화상회의

**실행**:
1. 다른 브라우저에서도 화상회의 입장
2. 두 사용자의 비디오가 동시에 표시되는 것 확인
3. 실시간 음성/영상 통신 확인
4. 화면 공유 기능 확인 (선택적)

---

## 7단계: 부가 기능 소개 (2-3분)

### 7.1 룰렛 기능

**실행**:
1. "룰렛" 탭 클릭
2. 룰렛 타입 선택 (음식, 장소, 활동)
3. 룰렛 실행 버튼 클릭
4. 결과 확인

### 7.2 리더보드 기능

**실행**:
1. "리더보드" 탭 클릭
2. 참여자별 점수 및 순위 확인
3. 점수 계산 기준 설명

### 7.3 설정 및 방 관리

**실행**:
1. "설정" 탭 클릭
2. 프로필 이미지 변경 기능 확인
3. 방 설정 옵션 확인
4. 방 나가기 기능 확인

---

## 8단계: 모바일 반응형 및 PWA (2-3분)

### 8.1 모바일 반응형 확인

**실행**:
1. 브라우저 개발자 도구에서 모바일 뷰로 전환
2. 모바일 화면에서 주요 기능들 확인
3. 터치 인터페이스 최적화 확인

### 8.2 PWA 기능 소개

**실행**:
1. PWA 설치 버튼 클릭
2. 앱 설치 프로세스 확인
3. 설치된 앱으로 실행
4. 오프라인 기능 확인 (선택적)
